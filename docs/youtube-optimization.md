# YouTube Optimization Plan — A Lyme Life (@alymelife)

Internal working doc (excluded from the deployed site via rsync `--exclude 'docs/'`).
Goal: turn the existing YouTube library into a search channel that bypasses the
site's near-zero domain authority and feeds qualified visitors to the booking page.
Grounded in the July 2026 GSC data (money queries — hyperthermia, Germany, cost —
already rank page 2–4 on the site; YouTube can rank them faster).

---

## How to apply (mechanics)

- **Where:** YouTube Studio → Content (videos) and → Playlists. Titles/descriptions/tags
  are edited there. This is operator-only; the website cannot change YouTube-side text.
- **Titles:** front-load the search phrase in the first ~55 characters (titles truncate).
  Channel name auto-appends — do NOT put "— Christina Carter" in the title; it wastes budget.
- **Descriptions:** the FIRST line (above the "…more" fold, ~150 chars) must contain the
  search phrase + a site link. Put the money link first.
- **Pinned comment:** after each edit, pin a comment with the site link + "Free call:
  https://alymelife.com/book.html". Pinned comments get seen even when descriptions don't.
- **Site cards are independent:** the playlist/video titles shown on the website are
  hardcoded HTML. Renaming a playlist/video on YouTube for SEO does NOT change the site.
  → Keep warm human titles on-site; use keyword titles on YouTube. Best of both.
- **After YouTube titles change:** sync the on-site `VideoObject` schema `name`/`description`
  to match (dev task) so Google sees one consistent title, not two.
- **No emojis** in descriptions (house style).

## On-site page strategy for the video library

**Do NOT create 15 individual pages** (one per Germany video). On a young YMYL site this
backfires: thin/doorway content, keyword cannibalization against `blog-germany.html` +
the keystone, and dilution of scarce domain authority. One strong page beats fifteen weak ones.

**DO concentrate into hubs, and always pair video with real written text:**
1. **Fold the "day by day" Germany playlist into `blog-germany.html`** (already ranks pos 15
   for "german lyme disease clinic"). Embed the playlist + add an experiential day-by-day
   text section. Strengthens the page that's already winning instead of competing with it.
   (Alternative only if you want a distinctly experiential URL:
   `/lyme-treatment-germany-day-by-day.html` — but never both covering the same ground.)
2. **Build ONE "Preparing for Lyme Treatment in Germany / abroad" page.** Prep/checklist
   intent is genuinely distinct from "should I go / what does it cost," so it won't
   cannibalize. Earns a standalone page IF filled with substantial unique text (not just an embed).

**It's never site OR YouTube — do both.** The video ranks on YouTube (no domain authority
needed); embedding it on a rich text page adds on-site engagement + funnel. Pair, don't choose.
